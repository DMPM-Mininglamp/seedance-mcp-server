# Seedance MCP Server 使用说明

## 🎬 服务器状态

- **本地服务器**: `http://localhost:3002/sse`
- **状态**: ✅ 运行中
- **模型**: Seedance 1.5 Pro (doubao-seedance-1-5-pro-251215)

## 📋 可用工具

### 1. submit_video_task - 提交视频生成任务

**参数**:
- `prompt` (必需): 视频内容描述
- `image_url` (可选): 参考图片URL，用于图生视频
- `generate_audio` (可选): 是否生成音频，默认true
- `draft` (可选): 是否使用样片模式（480p低分辨率），默认false

**示例**:
```json
{
  "prompt": "一只可爱的小猫在阳光下玩耍，画面温馨明亮",
  "generate_audio": true,
  "draft": false
}
```

**返回**: Task ID，用于后续查询状态

### 2. check_video_status - 查询任务状态

**参数**:
- `task_id` (必需): submit_video_task 返回的任务ID

**任务状态**:
- `queued`: 排队中
- `running`: 生成中
- `succeeded`: 成功（返回视频URL）
- `failed`: 失败
- `cancelled`: 已取消
